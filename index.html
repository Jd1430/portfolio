1] FACE EMOTION DETECTION

Project Overview:

  • Dataset: FER2013 Labeled Grayscale Images
  • Libraries: OpenCV, TensorFlow, Keras, scikit-learn, Seaborn, pandas, numpy
  • Algorithms: Haar Cascade Algorithm, Convolutional Neural Network
  • Input: Normal Image or Video Capturing
  • Output:

    Happy
    Sad
    Angry
    Surprised
    Fearful
    Neutral


Specifications:

The CNN picks out the important patterns and find simple patterns like edges or corners and ignores the less important details. It focus on the key features of the face(eye movements, mouth shape, eyebrow position, nose and cheeks). Pattern-finding process happens in multiple layers. Each layer looks for more complex patterns, like shapes and textures, by combining the simpler patterns found in earlier layers After several layers CNN decides what emotion is being shown in the face (like happy, sad, or angry) based on the patterns it has learned

Project Overview:

  • Dataset: Soil, Weather, Historical Crop yield and Market Price Dataset

3] SPEECH EMOTION DETECTION

Project Overview:

  • Dataset: RAVDESS Dataset
  • Libraries: OpenCV, TensorFlow, Keras, scikit-learn, Dash, Librosa, pydub, Seaborn, pandas, numpy
  • Algorithms: MFCCs, CNN, RNNs
  • Input: Video Files, Live Camera with Microphone
  • Output: Emotions visualized with bar charts and waveforms using a Dashboard


Specifications:

The System is designed to analyze and identify emotions from speech in video files, system extracts the audio and extracted audio is then processed to extract features like MFCCs using Librosa, which are fed into a deep learning model built with TensorFlow or Keras to classify emotions. The results are displayed on an interactive dashboard which showcases the original video alongside visual representations of emotion detection, including bar charts for emotion distribution and waveforms for emotion intensity over time, The System enables real-time emotion analysis from videos, which can enhance customer service by understanding emotional tones in interactions, assist therapists in assessing patients' emotional states for better diagnosis and treatment, and improve entertainment experiences by tailoring content based on viewer emotions.

4] CRICKET WIN% PREDICTOR

Project Overview

  • Dataset: Previous Cricket Matches Dataset
  • Input: Team's Current Score, Wickets and Number of Overs
  • Output: Win% of the Team


Specifications

The system is designed to forecast the probability of a team winning a cricket match by analyzing historical data, player statistics, and other relevant factors. It utilizes machine learning models to process data such as past match results, player performance metrics, pitch conditions, and team compositions. By applying algorithms like Logistic Regression, Random Forest, and Gradient Boosting, the system predicts win percentages and provides insights into match outcomes. This predictive capability aids teams, analysts, and enthusiasts in making strategic decisions and understanding match dynamics. The system offers a data-driven approach to enhance decision-making and strategic planning in cricket.
